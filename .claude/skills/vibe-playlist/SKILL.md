# vibe-playlist

参照アーティストまたは曲を起点に、Last.fm の類似データを使って未発見のトラックを集め、新しいプレイリストを作成して再生する。

## 入力パターン

| パターン | 例 | 起点 |
|---|---|---|
| アーティスト参照 | 「Radiohead みたいな落ち着ける曲」 | `get_similar_artists` |
| 曲参照 | 「この曲みたいな感じで」「Paranoid Android みたいなの」 | `get_similar_tracks` |
| 両方 | 「Oasis の Don't Look Back in Anger 系で」 | 曲優先 |

気分修飾語（「落ち着ける」「テンション上がる」「切ない」など）はプレイリスト名と説明文に反映するが、トラック選定のロジックには強く介入しない。

## 手順

### 1. 参照を特定する

ユーザーの発言から以下を抽出する：
- **参照アーティスト名**（あれば）
- **参照曲名**（あれば）
- **気分・シーン修飾語**（あれば：「落ち着ける」「朝に聴きたい」など）

曲名が明示されている場合は曲優先（ステップ2B）。アーティスト名だけの場合はステップ2A。

### 2A. アーティスト参照の場合

**ステップ 2A-1. 1段階目の類似アーティスト取得**

```
get_similar_artists(artist_name=参照アーティスト, limit=10)
```

返ってきた類似アーティストから **上位6〜8人** を選ぶ。

**ステップ 2A-2. 2段階目の類似アーティスト取得（並行実行）**

選んだ各アーティスト（6〜8人）に対して以下を実行し、各アーティストから **4〜5人の類似アーティスト** を取得：

```
get_similar_artists(artist_name=1段階目のアーティスト, limit=5)
```

1段階目と2段階目を合わせて、**重複を除外した20〜25人のアーティスト群** を集める。参照アーティスト自身は除外。

**ステップ 2A-3. トラック収集（並行実行）**

20〜25人のアーティストから、1アーティストあたり **1〜2曲** を選んで合計 **25〜30曲** を集める。各アーティストに対して：

```
search_tracks(query="artist:{アーティスト名}", limit=3)
```

選曲の基準：
- 曲名が代表的・知名度が高そうなもの（検索上位に来るもの）
- 同じアーティストから同名・同音源の重複は避ける
- 必要に応じて好みフィルタリング（テンポ、エネルギー感）を適用

### 2B. 曲参照の場合

```
get_similar_tracks(track_name=曲名, artist_name=アーティスト名（あれば）, limit=30)
```

返ってきた類似曲のリストから **25〜30曲** を選ぶ。

### 3. プレイリスト作成

収集したトラックURIを使ってプレイリストを作成する。

**名前の付け方：**
- アーティスト参照：`{参照アーティスト} vibes / {日付}`
- 曲参照：`Like "{曲名}" / {日付}`
- 気分修飾語がある場合は追加：`Yogee vibes - 落ち着ける / 2024-01-15`

```
create_playlist(
  name=上記のルールで命名,
  description="〇〇 をきっかけに発掘したプレイリスト。vibe-playlist で生成。"
)
```

続けて：

```
add_tracks_to_playlist(playlist_id=作成したID, track_uris=[収集したURIリスト])
```

### 4. 再生

```
play_context(uri=spotify:playlist:{作成したプレイリストID})
```

### 5. 出力フォーマット

```
## 🎵 [プレイリスト名]

[参照アーティスト/曲] をベースに [N] 曲を集めました。

**収録アーティスト：**
- [アーティスト名] — [曲名]
- [アーティスト名] — [曲名]
...

再生を開始します。
```

収録アーティストは全曲ではなく代表的なもの（8〜10組）を簡潔にリストするだけでよい。

## ポイント

- 参照アーティスト自身の曲は含めない（ユーザーはそのアーティストを知っているから）
- 類似アーティストが見つからない場合は `search_tracks(query="{アーティスト名} similar")` などで代替する
- プレイリストへのトラック追加は一度にまとめて行う（ループで1曲ずつ追加しない）
- 気分修飾語に強く縛られすぎず、まず「このアーティストに音楽的につながっている人」を優先する

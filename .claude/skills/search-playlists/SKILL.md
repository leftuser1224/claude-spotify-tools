# search-playlists

ユーザーのプロンプトから検索意図を読み取り、Spotifyのプレイリストを検索して結果を提示する。

## 手順

1. **検索クエリを組み立てる**
   - プロンプトから検索したい内容を抽出
   - 日本語の場合は英語クエリに変換（Spotifyは英語クエリの方が結果が豊富）
   - 必要なら複数クエリを試す

2. **検索実行**
   - `search_playlists(query, limit=5)` を実行
   - nullエントリを除外する

3. **出力フォーマット**

```
## 検索結果：「[クエリ]」

1. **[プレイリスト名]**（[曲数]曲）— by [オーナー名]
2. ...

流しますか？
```

4. **再生**
   - ユーザーが選んだら `play_context(uri)` で再生

## 注意
- 結果が少ない or nullだらけの場合は別のクエリで再検索する
- おすすめを求められている場合は `recommended-playlists` スキルを使うこと

# listener-profile

過去1年以上のSpotifyデータ × Last.fm音楽グラフを使って、「このリスナーはどんな人か」を多角的に描き出す。ジャンルの好み・開拓傾向・変遷の3軸で立体的なポートレートを作る。

## 手順

### 1. データ収集（並行実行）

- `build_taste_profile(time_range="long")` — 音楽DNA（ジャンル比率・上位アーティスト × Last.fmタグ）
- `analyze_discovery_rate()` — 新規開拓スコアとリスナータイプ診断
- `analyze_listening_evolution()` — 短期→長期のジャンル変遷とアーティストの動き

### 2. 3軸の読み取り

**軸1：音楽DNA（何を聴くか）**
- `genre_breakdown` から上位ジャンルを束ねてマクロジャンルに整理
- 地域（UK / 日本 / US等）・年代感（タグから推測）を付加
- 「このリスナーの音楽的ルーツ」として言語化する

**軸2：開拓スタイル（どう聴くか）**
- `discovery_score` を解釈する
  - 70以上 → 常に新しいものを探している冒険家
  - 40〜70 → バランスよく新旧を聴くExplorer
  - 40未満 → 信頼できる好みを深く聴くComfortable型
- `new_artists_recently` を見て「最近どこに向かっているか」を補足

**軸3：変遷（どう変わってきたか）**

ジャンル変化の分析：
- `genre_by_period` の long / medium / short それぞれのトップ10を比較する
- 各ジャンルの数値を long→medium→short と追いかけ、増減を具体的に示す（例：rock 17%→13%→10%）
- 増加したジャンルは「最近興味が広がっている方向」として解釈
- 消えたジャンル（long にあって short にない）は「一段落した時期」として言及
- 変化が小さいジャンルは「軸として定着している」と伝える

アーティスト変化の分析：
- `rising` アーティストをジャンルでグルーピングし、「どんな方向への開拓か」を読む（例：英国ロック史の掘り直し / フォーク方面への進出 など）
- `fading` アーティストも同様にグルーピングして「何が一段落したか」を言語化する
- `consistent` アーティストは「このリスナーの音楽的な軸」として改めて言及する
- rising と fading の対比から「音楽趣味の重心がどちらに動いているか」を一文で総括する

### 3. 出力フォーマット

```
## あなたの音楽リスナープロファイル

### 音楽DNA
[マクロジャンルの説明。地域・年代・テクスチャを含む2〜3文]

主なジャンル構成：
- [ジャンル]：XX%（[一言説明]）
- [ジャンル]：XX%
- [ジャンル]：XX%

核となるアーティスト（常連）：[consistent アーティスト列挙]

---

### 聴き方のスタイル
**[リスナータイプ名]**（開拓スコア：XX/100）
[discovery_score の意味を自分の言葉で解釈した1〜2文]

最近新たに聴き始めたアーティスト：[new_artists_recently から抜粋・ジャンル傾向を添えて]

---

### 変遷：今どこへ向かっているか

**ジャンルの推移**
| ジャンル | 長期 | 中期 | 短期 | 方向 |
|---|---|---|---|---|
| [ジャンル名] | XX% | XX% | XX% | ↑/↓/→ |
| ... | | | | |
※ 変化が±2%以上のジャンルを中心に記載。増加は↑、減少は↓、横ばいは→

[ジャンル推移から読み取れる音楽的重心の変化を3〜4文で。具体的な数値を引用しながら説明する]

**浮上してきたアーティスト（rising）**
[ジャンル・傾向でグルーピングして解説。「英国ロック史の掘り直し：The Smiths / The Stone Roses」のように束ねて意味づけする]

**一段落したアーティスト（fading）**
[同様にグルーピング。「何が一段落したか」を読み取って言語化する]

**変わらず聴き続けているアーティスト（consistent）**
[このリスナーの音楽的な軸として再提示。共通する特徴を一文で]

**総括**
[rising と fading の対比から「音楽趣味の重心がどちらに動いているか」を一文で]

---

### 一言でいうと
[このリスナーを1文で表現するキャッチコピー]
例：「UKインディーとJロックを行き来しながら、静かに新しい音楽を探し続けるExplorer」
```

## ポイント

- データを読み上げるのではなく、**「このリスナーはどんな人か」を語る**ように解釈する
- キャッチコピーは抽象的すぎず、ジャンル・スタイル・傾向を具体的に盛り込む
- ジャンル推移テーブルは変化が小さいもの（±1%以下）は省略してよい
- rising / fading はジャンル別にグルーピングして意味づけする。単純な列挙にしない
- プロンプトに「〇〇だと思うんだけど合ってる？」などの仮説があれば、データで検証してから答える
